APP ABSENSI

1. Administrator	-> Login, 
			   Kelola (CRUD) user Tata Usaha 
			   Kelola (CRUD) user Dosen 
			   Kelola (CRUD) user Mahasiswa 
			   Ganti Password 
			   Logout

2. Tata Usaha		-> Login
			   Kelola (CRUD) Mata Kuliah 
			   Mengganti status absen mahasiswa (Bila mahasiswa mengantarkan surat izin ke tata usaha, dan tata usaha memeriksa kevalidan surat izin tersebut)
			   Ganti Password
			   Logout

3. Dosen 		-> Login
			   Mengabsen Mahasiswa berdasarkan mata kuliah dan pertemuan
			   Ganti Password
			   Logout

4. Mahasiswa		-> Login
			   Melihat status absensi (Bila tidak hadir tapi ada surat izin, surat izin harus diantar ke Tata Usaha)
			   Ganti Password
			   Logout
